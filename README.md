# MyCadenza Documentation

Architecture, concepts and code review notes for [MyCadenza](https://mycadenza.app),
a personal daily task planner for iPhone and iPad — SwiftUI, SwiftData, CloudKit.

This repository is **public**. The MyCadenza app is developed by Christian Hoferer,
who works as a tax advisor (Steuerberater). Because client work and §203 StGB
(German professional confidentiality) apply in his professional context, the
documentation in this repository follows a strict **sterility rule**:

- No client names or identifiers
- No real task titles drawn from professional work
- No internal details about the firm, its clients, or its workflows
- All examples are generic ("a client", "a deadline", "a recurring task")

If a topic touches on potentially sensitive content during a working session,
it is discussed verbally or kept in a private local file (gitignored), never committed.
